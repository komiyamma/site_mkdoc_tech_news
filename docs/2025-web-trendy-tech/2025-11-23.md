### Next.js

ここ一週間でNext.jsに関する話題では、Next.js 15の新機能やReact 19との統合、Tailwind CSSとの組み合わせによる高速UI構築が注目を集めています。特に、Next.jsを活用したAIインターフェースやバックエンド分離の議論が活発です。

- [@wesley_bytegrad](https://x.com/wesley_bytegrad/status/1990356833157234912) は、Next.jsアプリに別途バックエンドが必要な理由を解説し、BFF（Backend-for-Frontend）パターンやMotiaバックエンドの利点を動画で紹介しています。
- [@Suryansh777777](https://x.com/Suryansh777777/status/1991378542777741330) が、Gemini 3 Proを使ってNext.jsでJarvis風HUDインターフェースを構築した実験を共有し、MediaPipeやThree.jsとの連携が話題に。
- [@1337hero](https://x.com/1337hero/status/1991649054816915764) は、Next.jsとTypeScriptを排除してPreact + HonoでFaster Chatを再構築し、オフライン対応の高速性を強調。
- [@mohamed_djoudir](https://x.com/mohamed_djoudir/status/1990279799223296332) が、Next.js 16とTailwind CSS 4を使ったAIアシスタント付きAdminダッシュボードを公開し、NestJSバックエンドとの統合が注目。
- [@insights_m_x](https://x.com/insights_m_x/status/1991748435645730932) は、2025年のフロントエンドツールとしてVercelを挙げ、Next.jsのデプロイを推奨。

### Biome

Biome（JS/TSのリンター/フォーマッター）に関する直接的な話題は少ないですが、ウェブ開発ツールとしての速度と互換性が言及されています。Rustベースの効率が評価されていますが、今週のX投稿では環境関連のバイオーム（biome）と混同されるものが多かったため、技術的な新着はウェブ検索から抽出。
