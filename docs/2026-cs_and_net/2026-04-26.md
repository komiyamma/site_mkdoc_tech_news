**ここ一週間のC#/.NET関連ニュースまとめ（2026年4月19日〜26日）**

この期間に公式ブログやX上で話題になった主な更新は、セキュリティ関連の緊急対応とLinux環境の新機能、開発Tipsの3つが中心です。古いもの（4月14日以前のservicing更新やPreview 3など）は除外しています。

- **.NET 10.0.7 Out-of-Band（OOB）セキュリティ更新（4月21日公開）**  
  深刻な脆弱性（CVE-2026-40372）とリグレッションを修正した緊急パッチです。プロダクション環境を使っている場合は早急に適用を推奨されています。  
  公式ブログ：https://devblogs.microsoft.com/dotnet/dotnet-10-0-7-oob-security-update/

- **Ubuntu 26.04（Resolute Raccoon）での.NET新機能（4月23日公開）**  
  .NET 10が公式アーカイブに収録され、-resoluteコンテナタグの利用、Native AOTアプリのビルド、.NET 8/9のbackports PPA対応などが追加されました。Ubuntuユーザーにとって実践的なアップデートです。  
  公式ブログ：https://devblogs.microsoft.com/dotnet/whats-new-for-dotnet-in-ubuntu-2604/

- **.NET Native AOTでNode.jsアドオンを作成する方法（4月20日公開）**  
  C# Dev Kitチームが実際にC++依存を排除し、C# + Native AOT + N-APIでNode.jsネイティブアドオンを構築した事例を紹介。LibraryImportやUnmanagedCallersOnlyの活用例が詳しく解説されています。  
  公式ブログ：https://devblogs.microsoft.com/dotnet/writing-nodejs-addons-with-dotnet-native-aot/
