### .NET Framework 3.5の展開変更（Standalone Deploymentへ移行）
新しいWindowsバージョン（Windows 11 Insider Build 27965以降）では、.NET Framework 3.5がWindowsのオプションコンポーネントから除外され、スタンドアロンインストーラーとして別途提供されるようになりました。  
レガシーアプリに影響する可能性があるため、開発者は注意が必要です。  
詳細: [devblogs.microsoft.com/dotnet/dotnet-framework-3-5-moves-to-standalone-deployment-in-new-versions-of-windows](https://devblogs.microsoft.com/dotnet/dotnet-framework-3-5-moves-to-standalone-deployment-in-new-versions-of-windows/)  
[@dotnet](https://x.com/dotnet/status/2022304766924034172)の投稿で公式アナウンスされ、[@richcampbell](https://x.com/richcampbell)なども関連トピックで言及しています。

### .NET 11 Preview 1およびC# 15の初公開（2月頃リリース）
.NET 11の最初のプレビューが公開され、C# 15の初期機能も含まれる形でスタートしました。  
- ランタイム/ライブラリ/SDKの改善、Monoランタイムの削減、AI関連強化などが予告されています。  
- 2026年11月のGA（一般提供）に向けた開発サイクル開始です。  
一部コミュニティではC#の文法複雑化に対する議論も見られますが、生産性向上を目指した更新です。  
詳細: [devblogs.microsoft.com/dotnet/dotnet-11-preview-1](https://devblogs.microsoft.com/dotnet/dotnet-11-preview-1/)  
関連報道: [devclass.com](https://www.devclass.com/development/2026/02/13/net-train-keeps-rolling-with-first-showing-of-2026-release/4090277)
