**ここ一週間のC# / .NET関連ニュースまとめ（2026年3月1日〜8日頃）**

### 主要リリース・アップデート
- **Model Context Protocol (MCP) C# SDK v1.0 正式リリース**（3/5頃）  
  AI/LLMとツール・リソースをセキュアに連携させるための公式C# SDK。認証サーバー自動発見の改善、ツール呼び出し時のアイコンサポート、最小権限原則のスコープ同意、長時間実行リクエストのポーリング対応、ASP.NET Core統合強化などが追加されました。AIエージェント開発で特に注目を集めています。  
  詳細：https://devblogs.microsoft.com/dotnet/release-v10-of-the-official-mcp-csharp-sdk/

- **JetBrains dotInsights March 2026 公開**（3/4頃）  
  Rider / ReSharper 2025.3.2 更新情報に加え、**VS Code / Cursor対応のReSharper正式リリース**をアナウンス。LSPベースでC#インスペクション・リファクタリングが利用可能になりました。  
  詳細：https://blog.jetbrains.com/dotnet/2026/03/04/dotinsights-march-2026/

- **C# 15 早期プレビュー機能「unions（判別共用体）」が .NET 11 Preview 3 にマージ**  
  `union Pet(Cat, Dog, Bird)` のような宣言と `[Union]` 属性、完全網羅性チェック付きパターンマッチングが利用可能に。C#ファンから大きな反響です。
