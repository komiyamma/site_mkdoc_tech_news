**ここ一週間（2026年3月8日〜15日）のC# / .NET関連ニュースまとめ**（X上の投稿およびインターネット情報に基づく）。

### 1. .NET 11 Preview 2 公開（3月10日）
.NET 11の第2プレビューがリリースされ、ランタイム・SDK・ライブラリ・ASP.NET Core / Blazor・.NET MAUIなどで改善がありました。  
主なハイライト：  
- Matrix4x4 が15%高速化  
- SDKインストーラーとコンテナイメージが最大17%小型化  
- 新しいC#機能（#elif、partitionWithなど）  
- Blazor Web Workerテンプレート追加  
- EF Coreにベクター検索サポート  

公式ブログで詳細確認可能。  
[@IntitechDev](https://x.com/IntitechDev/status/2031597646805148086) が性能向上や新C#機能を紹介、  
[@DOTNETDOKTOR](https://x.com/DOTNETDOKTOR/status/2031470291327570314) が全体の改善点を共有していました。

### 2. .NET / .NET Framework 2026年3月サービシング更新（3月10日）
.NET 10.0.4 / 9.0.14 / 8.0.25 の定期更新が公開。  
セキュリティ修正（CVE-2026-26127など .NET関連のセキュリティ機能バイパス脆弱性を含む）および非セキュリティ修正（Runtime、ASP.NET Core、Entity Framework Core、WPFなど）が含まれます。  
.NET Framework側は今回更新なし。  
公式ブログでリリースノート・ダウンロード確認可能。  
[@ErikEJ](https://x.com/ErikEJ/status/2031470758791102939) がブログを共有、  
[@aturart](https://x.com/aturart/status/2031731444515840504) も同様に案内していました。  
（この更新に含まれる .NET 10.0.5 OOB も参照）

### 3. .NET 10.0.5 Out-of-Band（OOB）リリース（3月12日）
.NET 10.0.4で発生した回帰バグを緊急修正。  
macOS（特にApple Silicon / ARM64）でVisual Studio Codeを使って .NETアプリをデバッグするとクラッシュする問題を解決。  
他のプラットフォームは影響なし。VS Codeユーザーは即時更新推奨。  
公式ブログでリリースノート・ダウンロードリンク確認可能。  
（上記サービシング更新内で言及）
