**過去1週間のC#/.NET関連ニュースまとめ（2026年3月29日〜4月5日）**

この期間の主な話題は、**C# 15**の新機能プレビューと**.NET 11 Preview 2**のリリースを中心に、開発者コミュニティで活発に共有されました。古いセキュリティ更新（3月10日頃のPatch Tuesdayなど）は除外し、最新情報のみを抽出しています。

### C# 15の新機能：Union Types（共用型）がプレビュー公開
4月2日に公式ブログで「**Explore union types in C# 15**」が公開され、注目を集めました。C# 15では閉じた型のセットを宣言可能になり、暗黙の変換や網羅的なパターンマッチングが強化されます。これにより、ボイラープレートを減らし、より安全で表現力豊かなAPI設計が可能になります。  
[@TrailheadTech](https://x.com/TrailheadTech/status/2039765000080744719)が「C# 15 brings union types... a clean win for safer, clearer APIs in .NET」と紹介し、開発者から好評でした。  
詳細は公式ブログ（https://devblogs.microsoft.com/dotnet/csharp-15-union-types/）で今すぐ試せます。

### .NET 11 Preview 2がリリース
.NET 11の第2回プレビューが公開され、パフォーマンス改善や各種ライブラリ更新が話題に。  
[@Mike_Preston17](https://x.com/Mike_Preston17/status/2039978621750673476)が「.net 11 preview 2 is out, with some perf improvements and MCP stuff」と速報を共有。開発者間で早期検証が始まっています。

### C# 14の拡張機能（Extension Membersなど）が再注目
C# 14で追加された拡張プロパティ・演算子・静的拡張が「2007年の拡張メソッド以来の大きな変更」と評価されています。  
[@ParticularSW](https://x.com/ParticularSW/status/2039693113409638600)が「Have you looked at C# 14's extension members yet?」と呼びかけ、コミュニティで活発に議論されました。

### その他の開発者向けアップデート
- **xaml.io**がブラウザ上でC#アナライザーとコード修正に対応予定を発表。インストール不要で.NET（C# + XAML）アプリをビジュアルデザイナーで開発可能になります。  
  [@xaml_io](https://x.com/xaml_io/status/2038933191969370249)がスクリーンショット付きで告知。
- **.NET Dailyニュースレター**（Issue #424）でC# 14の10の新機能、.NET 10のProgram.cs深掘り、Gemini CLIを使ったレガシーアプリ近代化などがまとめられ、開発者の日常学習に役立っています。  
  [@dotnewsletter](https://x.com/dotnewsletter/status/2039312673171763524)参照。
- コミュニティイベントとして、Northern Utah .NET Users Groupが「High Performance C#」を4月9日に開催予定。  
  [@NUNUGUSA](https://x.com/NUNUGUSA/status/2040575452478128569)。

全体として、C# 15 / .NET 11に向けた準備が着実に進んでおり、AI統合やパフォーマンス向上の動きが目立ちました。指定アカウント群からは該当期間にC#/.NET関連の投稿は確認されませんでした。最新情報は公式devblogsや上記X投稿を直接チェックしてください！
