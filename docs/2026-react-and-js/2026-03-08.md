### TypeScript 6.0 RCリリース（3月6日）
現行のJavaScriptコードベースで**最後のリリース**となり、次期TypeScript 7.0（Go言語完全移植でコンパイル10倍速化予定）の橋渡し版です。  
主な新機能・変更：  
- this-less関数の型推論改善  
- es2025サポート  
- Temporal API型追加  
- RegExp.escape  
- DOMのIterable/AsyncIterable対応  
- --stableTypeOrderingフラグなど  

多数の非推奨設定や破壊的変更も含み、将来移行を促す内容です。npmで`typescript@rc`を試せます。  
公式ブログ：https://devblogs.microsoft.com/typescript/announcing-typescript-6-0-rc/  

Xでは即座に共有され、  
[@yanavery](https://x.com/yanavery/status/2030300738954358935) が「Announcing #TypeScript 6.0 RC」とリンク付きで告知。  
[@topickapp_com](https://x.com/topickapp_com/status/2030291630159171772) が日本語で「Goネイティブポートへの移行を支援する新機能追加」とまとめています。

### React / React Nativeエコシステムの小規模アップデート
- **terminui v0.3.0**（3月2日）：ターミナルUIツールキットに**JSXレイヤー**を追加。React-likeな`Column/Row`プリミティブや`terminalDrawJsx`ヘルパー、strict TS型付きで記述性が大幅向上（パフォーマンスモデルは従来のdouble-buffer差分描画を維持）。  
  [@MrAhmadAwais](https://x.com/MrAhmadAwais/status/2028377792698777691) が詳細と新例（jsx-chatbot.tsxなど）を投稿。  

- **Inngest TypeScript SDK v4 beta**（3月3日）：ミドルウェアシステムを全面刷新（直感的なhooks追加）、イベントスキーマのランタイム検証強化、並列ステップ最適化とCheckpointingのデフォルト有効化で高速化。ログもPino風に改善。  
  [@inngest](https://x.com/inngest/status/2028922542661222652) 公式アカウントが「big one...」と新機能を列挙。  

- **React Native Enriched**関連更新（3月6日）：Software Mansion Labsの拡張ライブラリで、HTML normalizer強化によりペースト時のフォーマット保持が向上（LaTeX横スクロール対応など）。  
  [@G_Moskaliuk](https://x.com/G_Moskaliuk/status/2029934246211539104) が関連リポジトリと改善点を共有。
