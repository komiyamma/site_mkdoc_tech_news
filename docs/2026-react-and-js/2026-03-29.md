**ここ一週間のJavaScript/TypeScript/React関連の新たな情報まとめ（2026年3月22日〜29日頃）**

### TypeScript 6.0 正式リリース（3月23日）
TypeScript 6.0が正式リリースされました。これは**JavaScriptコードベースで構築された最後のメジャーバージョン**で、来るTypeScript 7.0（Go言語で完全ネイティブ実装予定）への橋渡しリリースです。  
主な変更点：
- メソッドの型チェック強化
- 新しい標準ライブラリ対応（Temporal API、Map#getOrInsert、RegExp.escape）
- Node.js向け新モジュール機能（#/プレフィックスによるパッケージインポート）
- 新デフォルト設定：`target: ES2025`、`--strict`
- 多数の非推奨項目（baseUrl、outFile、import assertions、module namespace、moduleResolution: node、target: ES5など）

大規模プロジェクトでの型安全性とパフォーマンスがさらに向上し、7.0への移行準備が整いました。  
公式発表：[@typescript](https://x.com/typescript/status/2036124240144216334)  
詳細解説：[@robpalmer2](https://x.com/robpalmer2/status/2036140215668982086)

### Node.js セキュリティリリース（3月24日）
Node.js 25.x / 24.x / 22.x / 20.x ラインでセキュリティパッチがリリースされました。  
TLSエラー処理、HTTP/2フロー制御、暗号タイミング漏洩、権限モデルバイパス、V8ハッシュテーブル弱点など**7件の脆弱性**を修正（一部はリモートDoS攻撃やプロセスクラッシュの可能性あり）。  
直ちにアップデートを推奨します。  
公式発表：[@nodejs](https://x.com/nodejs/status/2036558047535911031)

- **WebStorm 2026.1**（3月25日リリース）：サービス駆動のTypeScriptエンジン強化、React/Angular/Vue/Astro対応の大幅改善、現代的なCSS色空間サポートなど日常開発ワークフローが大幅に向上。

- **Oxfmt beta**（継続アップデート中）：Prettierと100%互換の軽量JS/TSフォーマッター。Tailwind CSS対応やインポート並び替えが安定し、移行コマンドも提供。高速・単機能志向のチームに注目されています。  
  発表：[@voidzerodev](https://x.com/voidzerodev/status/2037127599491600500)
