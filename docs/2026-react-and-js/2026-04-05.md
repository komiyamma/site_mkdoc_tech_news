**ここ1週間（2026年3月29日〜4月5日）のJavaScript/TypeScript/React関連の新情報まとめ**（古いものは除外）。

### 1. TypeScript 6.0 GAリリース（3月23日リリース、4月に入り話題拡大）
TypeScript 6.0が正式リリースされ、**TypeScript 7.0（Goネイティブ移植で10倍高速化予定）の橋渡し版**として位置づけられています。主な新機能・変更点：
- `es2025` target/lib対応（`RegExp.escape`、`Map.getOrInsert`/`getOrInsertComputed`、Temporal APIの型サポートなど）
- `#/`プレフィックスによるサブパスインポート対応
- `this`を使わない関数の型推論改善
- デフォルト値の変更（`strict: true`、`module: esnext`、`target: es2025`など）と多数の非推奨化

VS Code 1.114でもTypeScript 6.0対応が発表され、開発体験が向上。ITトレンドまとめでも「tsgoでコンパイル10倍速」と取り上げられました。

### 2. Vite 8.0リリース（Rolldown採用で大幅高速化）
Rust製新バンドラ**Rolldown**を導入し、開発サーバーと本番ビルドを統一。実プロジェクトでのレポートでは**Vite 6/7比で7〜12倍高速**、ファイルサイズも最小化されたとの声が相次いでいます。地味に革命的と評価されています。

### 3. Pretext：ブラウザ向け超高速テキストレイアウトライブラリが登場
React/ReasonML/Midjourneyフロントエンド作者の**chenglou**氏が開発したTypeScriptライブラリ。**DOM読み取りゼロ・リフローなし**でテキスト測定・レイアウトを500倍高速化（120fpsで数十万テキストボックス処理可能）。Unicode/RTL/絵文字完全対応で、FigmaやVercelエンジニアからも注目を集めています。デモが「CSS不要レベル」と話題沸騰中。
