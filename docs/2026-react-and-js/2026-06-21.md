### TypeScript 7.0 RCリリース（大きなトピック）
- **TypeScript 7.0 RC**が6月18日頃に公開されました。コンパイラをGo言語に**ネイティブポート**したことで、**TypeScript 6.0比で約10倍高速**化（parsing/type-checking/emitの並列処理、shared memory parallelism活用）。大規模コードベース（Bloomberg、Figma、Vercelなど）で実績あり。
- `--checkers` `--builders`などの新フラグで並列制御可能。--watchモードも強化。セマンティクスは6.0と同一で移行しやすい（side-by-side運用用の@typescript/typescript6パッケージも提供）。デフォルト設定の変更（strict: trueなど）あり。
