### 1. Next.jsとReact Server Componentsの緊急セキュリティアップデート（5月上旬〜中旬リリース、引き続き話題）
- Next.jsチームとReactチームが**13件のセキュリティアドバイザリ**に対応したパッチをリリース。DoS（サービス拒否）、middleware/proxy bypass、SSRF（Server-Side Request Forgery）、cache poisoning、XSSなどの脆弱性を修正。React Server Components関連のCVE-2026-23870（高重要度DoS）も含む。
- 影響を受けるバージョン：Next.js 15.x/16.x（App Router使用時）、react-server-dom-* 19.x系。
- **推奨対応**：**即時アップデートを強く推奨**。Next.js 15.5.18 / 16.2.6 へ更新。CloudflareなどのWAFも一部緩和ルールを追加したが、パッチ適用が最優先。

X上では[@KumbajiK](https://x.com/KumbajiK/status/2058390060337127525)が「Everyone's patching Next.js right now」と指摘し、Next.js依存の見直し議論も出ています。

### 2. TypeScript 7.0関連の動き（Goコンパイラ移行準備）
- TypeScript 7.0で**Go言語による新コンパイラ**が導入予定（約10倍高速化）。拡張APIサポートが当初ないため、WebStormなどがフォーク対応を準備中。
