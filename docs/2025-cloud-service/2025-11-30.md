### Firebaseの新情報（2025年11月23日〜30日）

Firebaseのこの一週間では、Data Connectの新機能追加やCrashlyticsのWebサポートが話題に上がりました。公式ブログではAI関連の加速ツールが言及されましたが、厳密に一週間内の新着は限定的。Xでは開発者コミュニティからの共有が多く、PushbaseのようなFirebaseライクなツールの紹介も見られました。

- Firebase Data Connectにイベントトリガーと生成型Admin SDKが追加され、サーバーサイドコードの実行が容易に。詳細は公式ドキュメントで確認可能。
- CrashlyticsのWebアプリサポートがRFC段階で議論中。Web開発者からのフィードバックを募集中。
- PushbaseというExpoベースのFirebase風プッシュ通知サービスが公開。Expoアプリユーザー向けに活用可能。
- バックエンドとしてFirebaseを活用したCRMツールの開発事例が共有され、ユーザー認証やリード管理の改善が強調された。
- Firebase StudioのGemini 3統合が提案され、アプリビルドの効率化が期待されている。
- 公式リリースノートではAI Logicのサーバープロンプトテンプレートが11月21日に追加されたが、一週間内のニュースとして関連記事で言及。

- v0のワークフローを活用したアクセシビリティチェックやコメントベースのプロンプト生成のアイデアが議論。

### Cloudflareの新情報（2025年11月23日〜30日）

Cloudflareはこの一週間でAgentsのアップグレードやWAFの強化が目立ちました。公式ブログではFLUX.2の統合とWAFペイロードロギングの改善が発表。XではCPU課金の変更やRadarの新機能が共有され、セキュリティとパフォーマンスの進化が焦点。

- Agentsにresumable streamingが追加され、長時間のAIチャットが中断されても継続可能。MCPの改善も。
- HTTP Origins insightsがRadarに追加され、Cloudflareとクラウドオリジン間のトラフィック監視が可能に。異常検知APIも。
- WAFのアップデートでCVE-2025-64446の認証バイパス脆弱性対策とPHP Wrapper Injection検知が強化。
- ContainersとSandboxesのCPU課金が使用分のみに変更され、コスト削減が可能。
- Black Forest LabsとのパートナーシップでFLUX.2 [dev]をWorkers AIに統合。画像生成のリアリズムが向上。
- WAFペイロードロギングの改善でアクションの理由が明確になり、ノイズ低減とシグナル向上。

Supabaseは$5B評価の資金調達が大ニュースで、企業契約を拒否した戦略が注目。コミュニティミートアップの開催や新機能の共有がXで活発。Test suiteの発表でテスト環境の改善が強調されました。

- $5B評価で$100M調達達成。企業契約を拒否し、オープンソースに注力した戦略が成功要因。
- コミュニティミートアップが12月から開始。スワッグやトーク、 surprise が予定。
- Supabase Test suiteが発表。高速スピンアップとフルアイソレーションでテスト環境が向上。
- 匿名サインイン機能でユーザーオンボーディングを摩擦なく実現。背景でセキュアに情報を保存。
- SQLエディタでテーブルやRLSポリシーをMD形式でエクスポートし、Codexレビューが可能。
- メール収集APIの統合とUIの改善が進行中。
