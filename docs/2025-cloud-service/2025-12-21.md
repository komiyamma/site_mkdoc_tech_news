### Firebase
この一週間でFirebaseに関する主な新情報は、Gemini 3 Flashの統合とFirebase Studioの機能強化が目立つ。Gemini 3 FlashはFirebase AI Logicを通じて利用可能になり、アプリ開発の高速化を支援する。Firebase Studioでは、Gemini 3.0を活用した生成機能が追加され、アプリのプロトタイピングが容易になった。また、2025年のYear in Reviewが公開され、Firebase Data Connectの一般提供やリアルタイム更新の進化が振り返られている。

- Gemini 3 FlashがFirebase AI Logicで利用可能に。速度重視のフロンティアインテリジェンスを提供し、コストを抑えつつアプリをスマート化。
- Firebase StudioでGemini 3.0を活用したランディングページ生成機能が強化。無料で即時デプロイ可能。
- 2025 Year in Review動画が公開。Firebase Data ConnectのPostgreSQL管理やリアルタイム更新の進展をまとめている。
- Firebase AI LogicのサポートモデルにGeminiモデルとImagenモデルが追加。モバイル/ウェブアプリから直接操作可能。
- Gemini 3 Pro Imageへの移行ガイドが公開。AngularとFirebaseを使った写真エディタの思考プロセス機能強化。

### Vercel (Vercel/v0を含む)
Vercelはv0のプロンプトガイドや新機能、CLIのアップデートを中心に動きが活発。v0ではNotionとの連携やAWSデータベースのネイティブサポートが追加され、開発ワークフローが向上。Vercel CLI 50.0.0ではコマンドの一貫性が強化された。また、Susan St. Ledgerの新社長就任や価格改定の議論も話題に。

- v0のプロンプトガイド公開。ビルド内容、ユーザー、デザイン制約を明確にすることで高速生成を実現。 [@v0](https://x.com/v0/status/2000946891467669745)のライブイベントでCPOがvibe codingをデモ。
- Notionとv0の連携開始。セキュアにワークスペース接続し、ダッシュボードやアプリ生成を効率化。
- Vercel CLI 50.0.0リリース。開発者体験向上とコマンド一貫性強化。
- AWSデータベース（Aurora PostgreSQLなど）のネイティブサポートをマーケットプレイスに追加。
- データエクスポート機能追加と動的プレフィックス導入で観測クエリとプレビューURLが改善。
- `vercel.ts`発売。Type-Safeなプロジェクト構成を動的にサポート。
- Susan St. Ledger前Salesforce社長がVercelに就任。ユーザー基盤2倍、v0の普及を強調。
- 価格改定の議論がRedditで活発。使用量増加の通知メールが話題に。 Vercelコミュニティのニュースキャッシュ（12/15）でv0関連のミートアップを報告。

### Cloudflare
Cloudflareの主なトピックは2025年のRadar Year in ReviewとImpact Reportの公開。AIの台頭、ポスト量子暗号化の採用拡大、DDoS攻撃の記録更新を分析。また、ネットワーク障害後のレジリエンス計画やストリーミング虐待対策の革新も発表。ダッシュボードのUI更新も行われた。

- 2025 Radar Year in Review公開。インターネットトラフィック19%増、AIクローラー増加、ポスト量子暗号化52%採用。
- 第5回Impact Report公開。選挙セキュリティ、ジャーナリズム保護、AIスクラッピング対策のマイルストーン。
- Code Orange: Fail Small計画発表。11/18のネットワーク障害後、レジリエンス強化で小規模失敗を重視。
- ストリーミング虐待対策の革新。H1 2025透明性レポートで詳細公開。
- ダッシュボード更新。サイドバー、ナビヘッダー、アカウントスイッチャー、Command+K改善。
- 2025開発者プラットフォーム更新。Node.js API拡張、多モデルAI検索、コンテナ拡大、リモートバインディング。
- WAF更新でReact RCE保護強化。サーバー関数虐待対策拡大。 [@Cloudflare](https://x.com/Cloudflare/status/2000712357064896816)で最大UDP DDoS攻撃検知を報告。

### Supabase
Supabaseの2025年統計がWrappedで公開され、ストレージや関数呼び出しの規模が明らかになった。新機能としてStripe Sync Engineの統合とVector Bucketsが追加。Amazon S3 Tablesとの連携も強化され、初心者向けガイドも登場。

- 2025 Wrapped公開。64.5 PBデータ保存、49.6B Edge Functions呼び出しなど。
- Stripe Sync Engine統合。ダッシュボードでStripeデータをSupabase DBに1クリック同期。
- Vector Buckets新ストレージオプション。S3の耐久性と類似検索を内蔵。
- Amazon S3 Tables統合。オープンウェアハウスアーキテクチャでETLストリーミング近リアルタイム。
- RLS機能のストレス要因分析記事。フロントエンド開発者向けルールガイド。
- 初心者向けガイド公開。BaaSプラットフォームとしてのツールセットを解説。
- PostgresWeekly 2025トップ5にSupabaseプロジェクト2件。 [@kiwicopple](https://x.com/kiwicopple/status/2002080558403498476)が維持を報告。
- xCloudでSupabaseサポート開始。SaaS/ウェブアプリ構築を簡素化。 [@ajee335](https://x.com/ajee335/status/2001003978289766658)がPySaaSに統合。

### Azure
Azureの12月更新がパートナー向けにまとめられ、AIツールやガバナンス機能が強化。Azure Developer CLIの拡張強化やSQL更新がGA。2025 Year in ReviewでSQL Server/Azure SQLの進展を振り返り、インフラの次世代計画も発表された。また、Windows更新によるRemoteApp問題が確認。

- 12月パートナー更新。AIツール、ガバナンス、クラウド革新でインテリジェントソリューション構築支援。
- Azure Developer CLI (azd) 12月更新。拡張強化、Foundryリブランディング、Azure Pipelines改善。
- Azure SQL 12月上旬更新GA。 [@sshzk](https://x.com/sshzk/status/2001460281562796350)と [@tomonari_q](https://x.com/tomonari_q/status/2001475965378269451)が報告。
- 2025 Year in Review: SQL Server/Azure SQL。移行、Managed Instance、Databaseの更新リスト。
- Azureインフラの次世代。Open Hardwareサーバー更新、Azure Boostアクセラレータ25%装備。
- Windows更新でAzure Virtual Desktop RemoteApp接続障害。Win11 24H2/25H2、Server 2025影響。
- Azure Architecture Center新着。クラウドアーキテクチャスタイル/デザインパターン更新。
- 週間Azure Update（12/19）。Functions Service Bus Trigger、ANF CZRRなど。 [@NTFAQGuy](https://x.com/NTFAQGuy/status/2002046809766797502)が動画公開。
