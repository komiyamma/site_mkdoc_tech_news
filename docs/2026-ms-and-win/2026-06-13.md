**ここ1週間（2026年6月6日〜13日頃）のMicrosoftおよびWindows 11に関する主な情報まとめです。**

### Windows 11 June 2026 Patch Tuesday更新（KB5094126）のリリース
最も大きなトピックは、**6月9日（Patch Tuesday）にリリースされた累積更新KB5094126**（OS Build 26200.8655 / 26100.8655、対象：24H2 / 25H2）です。約200件（記録的な数）の脆弱性を修正し、セキュリティ強化に加えて性能向上や新機能が複数含まれています。

主な改善点・新機能：
- **Low Latency Profile**: アプリ起動や操作応答を高速化。CPUを短時間で素早く動作させ、体感待ち時間を短縮。
- **CPU Boost / 性能向上**: 特に古いPCや中程度スペック機での速度・安定性改善。
- **共有オーディオ（Bluetooth LE対応）**: 複数のデバイスでオーディオ共有しやすく。
- **Multi-App Camera**: 複数のアプリが同時にカメラを使用可能。
- Windows Helloの改善、Secure Boot証明書の更新（古い証明書期限対応）、フォルダ名カスタマイズ、検索改善など。

**インストール方法**: Settings > Windows Update > Check for updates で適用可能。オフラインインストーラもMicrosoft Update Catalogで入手可。
