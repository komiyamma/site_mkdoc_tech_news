ここ一週間（2026年3月8日〜15日）のMicrosoftおよびWindows 11に関する主な情報は、**月例のセキュリティ更新プログラム（パッチチューズデー）**のリリースが中心でした。大きな新OS発表などはなく、セキュリティ強化と一部機能改善が目立ちました。古い情報は一切除外しています。

### 1. Windows 11 3月セキュリティ更新プログラムのリリース（3月10日米国時間／11日日本時間）
Microsoftは全サポート対象Windows 11バージョン向けにセキュリティ更新を公開。**CVE番号ベースで84件の脆弱性**を修正（うち一部はゼロデイ攻撃が公開済みで、Criticalレベルのものが8件含まれる）。特に**.NET**や**SQL Server**、**Office/Excel**関連の致命的な問題も対応されました。

- **主なKB番号**（Windows 11向け）：
  - 25H2 / 24H2：**KB5079473**（OS Build 26200.8037 / 26100.8037）
  - 26H1：**KB5079466**
  - 23H2：**KB5078883**

**新機能・改善点**（主に25H2/24H2向け）：
- Emoji 16.0サポート
- タスクバー／ネットワーク内蔵スピードテスト
- File Explorerの複数ドライブ検索信頼性向上
- WebP形式のデスクトップ背景サポート
- Windows Updateのサービススタック高速化
- Armデバイス向けRSAT（リモートサーバー管理ツール）サポート
- Secure Boot証明書の自動更新対象拡大
- Sysmon（システム監視ツール）組み込み強化
- Quick Machine Recoveryの改善

[@TechKard](https://x.com/TechKard/status/2032445817165066499)さんは、KB5079473で「WebPデスクトップ背景、File Explorer検索改善、組み込みスピードテスト、Arm向けRSAT」などが追加されたと詳細に投稿しています。  
[@Barnacules](https://x.com/Barnacules/status/2031607530049384601)も「Patch Tuesdayで9つの新機能＋セキュリティパッチ」と動画付きで紹介。

一方、適用後に**インストール失敗・再起動ループ・フリーズ・グラフィックス破損**などの報告が相次ぎ、Microsoftは調査中です。また、SamsungノートPC一部モデルで「Drive C（システムドライブ）がロックされる」深刻バグ（2月更新が引き金だが今週確認）が発生し、**緊急修正KB5084597**もリリースされました。

[@NeowinFeed](https://x.com/NeowinFeed/status/2032968535576576086)さんは今週のMicrosoft Weeklyまとめで「Windows 11をcrippleする深刻バグと緊急更新」を取り上げています。

### 2. Windows Insider Program（プレビュー版）の動き
Release Preview／Dev Channelなどで追加ビルドが展開：
- NarratorにCopilot統合強化
- Smart App Controlのトグル追加（信頼性向上）
- 極端なリフレッシュレートや指紋リーダーの信頼性改善
- 初期セットアップ新機能など

[@windowsforum](https://x.com/windowsforum/status/2032161745746174172)さんは「AIポリッシュ＋Smart App Controlトグルでより洗練された体験」と投稿。

### その他のMicrosoft関連
Microsoft 365 Copilotの小規模機能追加（エージェント強化など）はロードマップに記載されましたが、今週の目玉はWindows 11本体更新でした。公式サポートページや**窓の杜**、Microsoftセキュリティブログで早期適用を強く呼びかけています。

ユーザー投稿では更新後の不具合報告が散見されましたが、全体として「セキュリティ最優先の安定化月」でした。適用はWindows Updateから早めにどうぞ！
