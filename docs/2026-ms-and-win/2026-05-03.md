**ここ一週間（2026年4月26日〜5月3日頃）のMicrosoftおよびWindows 11関連の主な情報まとめです。**  
古いニュース（4月中旬以前のPatch Tuesday本体など）は除外し、最新のプレビュー更新やユーザー影響の話題を中心に、WebニュースとX上の最近の発言から抽出しました。

### 1. Windows 11 非セキュリティプレビュー更新（KB5083631）のリリース
- 4月30日頃にWindows 11（26H1/25H2/24H2向け）で**オプションのプレビュー更新**が公開されました。5月12日頃に正式ロールアウト予定。
- **主なハイライト**:
  - **Xbox Mode**の正式対応：PC（ノートPC・デスクトップ・タブレット）でフルスクリーンゲーム中心インターフェース。XboxアプリやWin+F11で起動可能。
  - ハプティックフィードバックの強化（PowerPointオブジェクト配置時など）。
  - スタートアップアプリ起動パフォーマンス向上。
  - Explorer.exeの信頼性向上、AIエージェントサポート強化、バッチファイルセキュリティ強化など34件以上の変更。

X上でも話題：
- [@nextgenos2026](https://x.com/nextgenos2026/status/2050156340463513606) がXbox Modeやhaptic feedbackを強調。
- [@XavierRiveraX](https://x.com/XavierRiveraX/status/2050156340463513606) が「34 changes、Xbox mode on every PC」と紹介。

### 2. Windows Updateのユーザー制御強化
- Microsoftが**更新の無期限延期**やインストール時のスキップを可能にする変更をテスト中。強制再起動の減少や更新バンドル化で中断を最小化。
- ユーザーから「ようやく待望の機能」と好評の声。

X上：
- [@HoltzJholtz111](https://x.com/HoltzJholtz111/status/2050829229663682719) が「updates to be skipped during install or even paused indefinitely」と共有。
- [@hogatoga](https://x.com/hogatoga/status/2050828286994080039) も同様に報じています。

### 3. Aprilセキュリティ更新（KB5083769など）の影響（継続報告）
- 一部ユーザーで**サードパーティ製バックアップアプリ（Acronis、Macrium Reflectなど）が失敗**する問題が発生。Volume Shadow Copy Service（VSS）のタイムアウトや脆弱ドライバブロック（psmounterex.sys）が原因。
- Microsoftは「更新をアンインストールせず、アプリ側最新版を使用」と対応を説明。BitLocker回復キー表示の可能性も指摘。

X上では直接的な新投稿は少なかったが、関連で不満の声（Windows 11全般）が見られます。

### 4. その他の最近の動き
- **Windows Insider Program**の刷新継続：パフォーマンス・信頼性向上、チャネル整理。
- **Secure Boot証明書更新**：デバイスセキュリティアプリで「Action is Needed」警告が出る場合あり。5月中旬にフェーズ2で黄色/赤警告強化。
- 全体としてMicrosoftは「Windows 11の品質向上（K2計画？）」を推進中。ネイティブアプリ回帰やExplorer改善も進行。
