**過去1週間（2026年3月15日〜22日）のMicrosoftおよびWindows 11主な話題は2つです。**

### 1. Windows 11向け緊急Out-of-Band更新「KB5085516」リリース（3月21日）
3月のPatch Tuesday更新（KB5079473）適用後、一部のユーザーでMicrosoftアカウントを使ったアプリ（OneDrive、Teams Freeなど）のサインイン時に「インターネット接続なし」エラーが発生する問題を修正する緊急パッチです。インターネットが正常でも影響が出ていましたが、企業向けMicrosoft Entra ID認証は対象外です。  
対象はWindows 11 24H2（ビルド26100.8039）および25H2（ビルド26200.8039）で、Windows Updateの「オプションの更新」から入手可能、手動ダウンロードもMicrosoft Update Catalogやサポートページからできます。

X上でも即座に話題に上がり、  
[@deskmodder](https://x.com/deskmodder/status/2035610076581433764) が「KB5085516 Windows 11 25H2 / 24H2 Out of Band Update」と詳細を共有、  
[@nekodaisuki15](https://x.com/nekodaisuki15/status/2035609720019681623) はYouTubeで更新内容を解説、  
[@ryo_iv](https://x.com/ryo_iv/status/2035606264311636366) は日本語サポートページリンクとともに「Microsoftアカウントを使っていると…のバグフィックス」と指摘していました。

### 2. ユーザー反馈を受けてのWindows 11大規模改善計画をMicrosoftが正式発表（3月21日頃）
Windows & Devices担当社長Pavan Davuluri氏が公式ブログなどで、Feedback Hubの声に応じた2026年の抜本的品質向上を明らかにしました。Insider Previewで3〜4月から順次テスト開始、年内広範展開予定です。主な変更点は以下の通りです。

- **タスクバー**：トップ・左右配置が可能に（長年の要望No.1）
- **Windows Update**：セットアップ時のスキップ、再起動/シャットダウン時の強制回避、長期間の一時停止、強制再起動の大幅削減、通知減少、月1回の再起動へ移行
- **Copilot**：Notepad・Photos・Snipping Tool・Widgetsなどで表示を意図的に減らし、プロモーションも抑制
- **File Explorer & パフォーマンス**：起動高速化、検索・ナビゲーション遅延低減、メモリ使用量削減、WebView2からWinUI3への移行でUIスムーズ化、ファイル転送・インデックス改善
- **その他**：ドライバ・Bluetooth・USBの信頼性向上、通知全体の削減

Xでは即反応があり、  
[@tbreakdotcom](https://x.com/tbreakdotcom/status/2035211893757849654) が「Windows 11 just got the overhaul users wanted: movable taskbar, less Copilot interference, optional updates」とまとめ、  
[@Mangu_Rants](https://x.com/Mangu_Rants/status/2035394877467361542) は「Movable taskbar confirmed」「pause Windows Updates for as long as you want」と具体的に列挙、  
[@Anurag9793](https://x.com/Anurag9793/status/2035333674582122942) も「Microsoft admits Windows 11 issues ⚠️ … movable taskbar, fewer forced updates, faster File Explorer & reduced Copilot」とユーザー感情を反映した投稿をしています。

この1週間は「不具合即時修正」と「長年の不満解消に向けた本気度」が目立つ内容でした。Windows 11 26H1（新ハードウェア向け）は継続情報のみで新展開なし。詳細はMicrosoft公式サポートページやWindows Insiderプログラムで確認してください。
