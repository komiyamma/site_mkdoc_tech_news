**ここ1週間（2026年5月10日〜17日頃）のMicrosoftおよびWindows 11に関する主な動きをまとめます。** 主にPatch Tuesday更新、新機能テスト、Edge関連の変更を中心に、最新情報に絞っています。

### 1. May 2026 Patch Tuesday更新（5月13日頃リリース）
- **KB5089549**（Windows 11 24H2/25H2向け）など、120件以上の脆弱性を修正（うち17件Critical）。ゼロデイはなし。セキュリティ更新として重要ですが、一部ユーザーでインストール失敗（ロールバック繰り返し）やインストール後のインターネット速度低下が報告されています。Microsoftは公式に広範な問題を認めていませんが、Redditやフォーラムで散見。
- Release Previewチャネルでは新ビルド（例: 26200.8514など）が提供され、品質向上の進捗が報告されています。

### 2. Windows 11のパフォーマンス・UI改善テスト
- **Low Latency Profile（macOSライクな速度ブースト）**: CPUを短時間高周波数に引き上げ、スタートメニューやアプリ起動を高速化する機能がテスト中。Microsoft副社長のScott Hanselman氏が「Appleもやっている」と擁護する声あり。
- **タスクバー位置変更・Startメニュー調整**: Experimentalチャネルでテスト開始。タスクバーを上下左右に移動可能、小型化オプション、Startメニューの「Pinned/Recent/All」セクション表示切替、プロフィール非表示などが追加予定。ユーザー体験向上を目指した変更です。

### 3. Microsoft EdgeのCopilot関連変更（5月13-14日頃）
- **Copilot Modeを廃止**し、AI機能をブラウザ本体に直接統合。複数タブ横断検索・比較・要約などがデスクトップ/モバイル両方で使いやすくなりました。専用モードは不要に。
