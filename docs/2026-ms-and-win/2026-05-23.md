**ここ1週間（2026年5月16日〜23日頃）のMicrosoftおよびWindows 11に関する主な動き**を、X（Twitter）とウェブ上の最新情報からまとめます。古い情報は除外しています。

### Windows 11の更新と新機能
- **26H1大型アップデート（Release Preview）**：Build 28000.2176で**Xbox Mode**（コンソール風体験）、File Explorerの改善、さまざまな修正が導入されました。
- **タスクバーとStartメニューの大幅変更**：タスクバーを画面の任意の位置に移動可能に。マルチモニターごとの位置設定や小型レイアウト、カスタマイズ強化がInsider Previewでテスト中です。
- **検索機能の改善**：ローカルアプリ・ファイル優先表示を強化（ウェブ結果の押し売り減少）。スペルミス時も正確性が向上。
- **パフォーマンス向上**：**Low Latency Profile**機能でCPUクロックを短時間ブーストし、Startメニューやアプリ起動を高速化するテストが進んでいます。

### セキュリティとパッチ
- **May 2026 Patch Tuesday**（KB5089549など）：120件の脆弱性を修正（Critical 17件）。ただし、一部ユーザーでインストール失敗やインターネット速度低下の報告あり。Microsoftが問題を認識しています。

### 新ハードウェア（Surface）
- **新Surface for Business**：Surface Pro / Laptop for BusinessにIntel Core Ultra Series 3プロセッサ搭載。AI加速とビジネス向け性能を強化し、5月19日頃から一部市場で提供開始。

### その他のMicrosoft動向
- CopilotのOffice統合強化（Word/Excel/PowerPoint）や、企業向けAI展開（Accenture、Moody’sなど）の事例が複数発表されました。
- Windows Insider Programのチャネル整理が進み、新ビルド画像もリリースされています。
