### マイクロソフトとWindows 11のここ一週間の主な情報まとめ

ここ一週間（2025年12月14日から12月21日）のマイクロソフトとWindows 11に関する情報を、X（旧Twitter）のユーザー投稿やインターネット上のニュースから集めました。主に最新のInsider Previewビルドのリリース、セキュリティアップデートの問題点、AI機能の進展、手持ちデバイス向けの最適化を中心に、古いニュースを排除してまとめています。情報は多角的に収集し、Xの引用はアカウントを分散させています。

#### 1. Windows 11 Insider Previewの新ビルドリリース
マイクロソフトは、Windows 11の開発チャネルで複数の新ビルドを公開しました。これらは主に機能向上と修正を目的としたもので、Insiderプログラム参加者向けです。

- **Dev & Beta Channels: Build 26220.7523 (KB5072043)**  
  12月19日にリリースされたこのビルドは、タスクバーに「Ask Copilot」機能の追加、新たなAIエージェントのサポート、タスクバーの改善（例: モダンなRunボックス）、および多数の修正を含みます。Xでは、[@winaero](https://x.com/winaero/status/2002098673447227788)が「Ask Copilot on the taskbar, New Agents, Agent Launchers」など新機能の詳細を共有しています。 また、[@Pureinfotech](https://x.com/Pureinfotech/status/2002078594227712025)がこのビルドのスクリーンショットを投稿し、プレビュー品質のアップデートとして紹介しています。 ニュースサイトでは、Microsoft公式ブログが詳細を発表し、システム全体でのAIエージェントの発見しやすさを強調しています。

- **Canary Channel: Build 28020.1362**  
  12月16日に公開された26H1ビルドで、ゲーム機能の強化（ハンドヘルドデバイス向けの電源管理やグラフィックス最適化）、File ExplorerのUI改善（完全ダークモードの適用、クラウドファイルの高速アクセス）、Settingsアプリの修正が主な内容です。[@itstechbased](https://x.com/itstechbased/status/2000961451696021992)が動画リンク付きで新機能を解説しています。 また、[@MrTbone_se](https://x.com/MrTbone_se/status/2000839360523198851)がMicrosoftブログからの新着情報として共有し、ITプロ向けのヒントを追加しています。 ニュースでは、Neowinが大量の新機能リストを報じています。

#### 2. セキュリティアップデート（KB5072033）の問題点
12月9日にリリースされたDecember 2025セキュリティアップデート（Windows 11 25H2/24H2向け）が、一週間内で議論を呼んでいます。主な問題はVPN接続の破損とWSL（Windows Subsystem for Linux）のバグです。

- Microsoftはアップデート適用後にVPNが動作しなくなる問題を確認し、修正を約束しました。Forbesがこれを報じ、「もう一つのWindowsアップデートによる機能破損」と指摘しています。 Xでは、[@DbAshby](https://x.com/DbAshby/status/2000467237119909899)がNeowinの記事を引用し、セットアップとリカバリーの更新（KB5072537など）を紹介しています。
- また、WSLのエラーや他のエッジケースが報告されており、Windows Latestが安定性を認めつつ、特定ユーザーの問題をまとめています。 Microsoftのサポートページでは、25H2バージョンの既知の問題リストを更新中です。

#### 3. AI機能とプライバシー関連の進展
Windows 11のAI統合が進んでおり、一週間内でいくつかの発表がありました。

- AIエージェントが個人ファイルにアクセスする前に同意を求める設定をMicrosoftが確認。Xの[@Pirat_Nation](https://x.com/Pirat_Nation/status/2002272762837241874)がこれを報じ、ユーザーの反発を指摘しています。 これはプライバシー懸念への対応です。
- Copilotのボイスモードで言語設定が可能になる新機能が開発中。[@testingcatalog](https://x.com/testingcatalog/status/2002528763960254973)がスクリーンショットを共有しています。
- 全体として、Windows 11を「AI-native」OSとして再構築する方針が強調されており、[@Coachvelozo](https://x.com/Coachvelozo/status/2000667410337046842)が動画でローカルAI実行の利点を説明しています。

#### 4. その他のトピック
- **リソース消費の変化**: アップデート後、AppX Deployment Serviceがスタートアップ時に実行され、CPU/RAM/ディスクの負荷が増す可能性。NeowinとBetaNewsがこれを報じ、25H2/24H2への影響を警告しています。
- **ゲーム最適化**: ハンドヘルドPC向けの電源管理やシェーダー配信の改善をMicrosoftが約束。[@Nafty1115](https://x.com/Nafty1115/status/2000859932787376629)がWccftechの記事を引用しています。
- **ユーザー体験の改善**: File Explorerのダークモード完全対応やポップアップ位置変更が話題。[@Affinity_IT](https://x.com/Affinity_IT/status/2001978489793073202)がこれをまとめています。
