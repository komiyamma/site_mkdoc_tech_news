### JavaScript/TypeScript/React のここ一週間の新たな情報まとめ

2025年12月7日から12月14日までの最新情報を、X（旧Twitter）の投稿とウェブ検索結果から集めました。主なトピックは、TypeScript 7の進捗、React Server Componentsのセキュリティ脆弱性パッチ、JavaScriptの30周年、関連ライブラリの更新などです。古いニュースは排除し、最近の動向に焦点を当てています。Xの引用は多様なアカウントから分散して選び、各リンクは具体的な投稿に直結させています。

#### 1. TypeScriptの進捗と新機能
TypeScriptチームは、12月上旬に「TypeScript 7」の進捗を発表。ネイティブコードへのポート（コードネーム: Corsa）が進み、オートインポート、find-all-references、リネームなどの機能が実装され、大規模コードベースでの日常使用が可能になりました。TypeScript 6.0を最後にJavaScriptベースのコードを終了し、ネイティブ版への移行を加速させる方針です。これにより、パフォーマンスとメモリ効率が向上します。

- X上で、TypeScript 7の「Corsa」コードネームと形状形成について議論され、セキュリティニュースと並行して注目を集めています。[@andrisweb](https://x.com/andrisweb/status/1999370776172957820)
- また、TypeScriptのPRがマージされ、`prop: function() {}`と`prop: () => {}`の動作差異が解消。開発者が混乱しにくくなりました。[@AndaristRake](https://x.com/AndaristRake/status/1998840168158015994)
- ネイティブサポートがすべての主要JavaScriptランタイムで実現し、TypeScriptの採用がさらに広がっています。[@ZamDeveloper](https://x.com/ZamDeveloper/status/1998067197462167930)

#### 2. Reactのセキュリティとアップデート
React Server Components（RSC）に深刻なリモートコード実行脆弱性「React2Shell」（CVE-2025-55182）が発見され、12月3日に公開。ReactとNext.jsの即時アップデートが推奨され、8日間で3つのCVEがパッチされました。これにより、サーバー関数が悪用されるリスクが軽減されています。

- React 19.0から19.2へのアップデートでは、基盤シフトが進み、パフォーマンス向上と新機能（Concurrent pre-rendering、Instrumentation APIなど）が追加。開発者はこれらを活用してアプリを最適化できます。
- X上で、React Router 7.9の更新が共有され、ミドルウェアと型安全コンテキストAPIが安定。React 19のトランジションサポートも強化されています。[@brophdawg11](https://x.com/brophdawg11/status/1998840627925233820)
- React Navigation v6 with TypeScriptの記事が共有され、ナビゲーションの型安全性が向上。[@inPlainEngHQ](https://x.com/inPlainEngHQ/status/1997737771331355020)

#### 3. JavaScriptの全体動向と関連ライブラリ
JavaScriptが正式に30周年を迎え、ReactのRSCセキュリティホール修正やTypeScript 7の進展がJavaScript Weeklyで取り上げられました。Node.jsの脆弱性パッチも並行して報告され、エコシステム全体のセキュリティ強化が進んでいます。

- X上で、TanStack AI SDKの新リリースとReact関連ニュースがまとめられ、React Native 0.83の<Activity>コンポーネントやReanimated 4.2のShared Elements Transitionsが注目。[@sebastienlorber](https://x.com/sebastienlorber/status/1998848328142901404)
- better-t-stackの更新で、Convexバックエンドのオプション化やTypeScriptカタログインストールが追加。[@amanvarshney01](https://x.com/amanvarshney01/status/1999811855603438075)
- Effect TSのウィークリーアップデートで、Effect Instituteの発表とDrizzleORMのネイティブEffectドライバ開発が共有。TypeScriptのスケーラビリティを強化しています。[@EffectTS_](https://x.com/EffectTS_/status/1999558355220590700)

これらの情報は、開発者がセキュリティパッチを優先しつつ、新機能で生産性を向上させるチャンスを示しています。詳細は各リンク先で確認してください。
