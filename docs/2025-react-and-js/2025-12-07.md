### TypeScript の最新情報

- **TypeScript 7.0 の進捗更新**：Microsoft は TypeScript 7.0 のネイティブポートの進捗を発表。チェック機能が TypeScript 5.9 と互換性が高く、言語サーバーの成熟度が向上。JavaScript ダウンレベルは ES2021 に制限され、将来的に ES2015 をサポート予定。API アクセスはまだないが、10倍の速度向上を実現。試用可能。 また、TypeScript 6.0 が JavaScript ベースの最後のメジャーリリースとなり、TypeScript 7 のネイティブツールチェーン (Corsa) がテスト可能で、ビルドが7.5〜10倍高速化。 X 上では [@robpalmer2](https://x.com/robpalmer2/status/1995916259125350764) がこのニュースを共有し、興奮を伝えています。[@typescript](https://x.com/typescript/status/1995913264467976546) の公式アカウントも詳細を投稿。[@iutkarsh077](https://x.com/iutkarsh077/status/1997197816402215387) は TS-first エコシステムの勢いを指摘。

- **AWS Strands Agents の TypeScript サポート**：AWS が Strands Agents SDK に TypeScript サポートを追加 (プレビュー)。エッジデバイス対応も発表され、モデル駆動型アプローチでエージェント構築を容易に。 X で [@adi_12_modi](https://x.com/adi_12_modi/status/1996259169004109878) が新機能を紹介し、[@kyhayama](https://x.com/kyhayama/status/1996497034548052173) が公式発表を共有。[@111330118y_y](https://x.com/111330118y_y/status/1996256583618335204) も TypeScript 版の可能性を言及。

### React の最新情報

- **React Server Components の重大脆弱性 (CVE-2025-55182)**：React と Next.js にリモートコード実行 (RCE) の脆弱性が発見され、CVSS スコア 10.0。React Server Components を使用するアプリが影響を受け、即時更新を推奨。Chrome 拡張機能経由の悪用が確認され、30組織が侵害、77,000 IP が脆弱。 修正ツールとして npx fix-react2shell-next や neurolint を使用。X で [@steren](https://x.com/steren/status/1997450790395576749) が Google Cloud ユーザーに警告を発信。[@sneakymonk3y](https://x.com/sneakymonk3y/status/1996889267402903605) が POC ペイロードのリリースを報告し、[@samuelmalkasian](https://x.com/samuelmalkasian/status/1997425010558878092) が即時更新を呼びかけ。[@catnap707](https://x.com/catnap707/status/1997444132596580524) は Cloudflare の障害との関連を指摘。

- **Formisch for React のリリース**：Solid.js 向けに作られたフォームライブラリ Formisch が React にポート。100% TypeScript で、バンドルサイズ 2.5kB から、型安全なフォーム作成が可能。ドキュメントは週末公開予定で、プレイグラウンドが利用可能。[@FabianHiller](https://x.com/FabianHiller/status/1996624573215789141) がリリースを発表。

- **Next.js 16 のリリース**：Cache Components で精密キャッシュ、Turbopack がデフォルト (2-5倍高速)、React 19.2 機能対応。AI 駆動 DevTools やスマートルーティングが追加。 X で [@prasanth__p_](https://x.com/prasanth__p_/status/1996232147993325605) がアップグレードの利点を強調。
