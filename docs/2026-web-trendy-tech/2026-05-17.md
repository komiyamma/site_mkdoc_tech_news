### Next.js
Next.jsでは**5月上旬のセキュリティリリース**が最大の話題。13件のアドバイザリ（DoS、middleware/proxy bypass、SSRF、cache poisoning、XSSなど）を修正。React Server Components関連のCVE-2026-23870も含む。Vercel推奨アップデート版は**15.5.18 / 16.2.6**。自ホスト環境は早急対応を。

Xでは実践例が増加：
- [@Dev_JesseMaduka](https://x.com/Dev_JesseMaduka/status/2055788449504894991): 「Client said “use Next.js” so I had to lock in... File-based routing, Better performance, Cleaner structure」とクライアントプロジェクトで好評価。
- [@athydev](https://x.com/athydev/status/2055830185329520746): Rails API + Next.js frontendでポートフォリオ作成の進捗報告。

一部で「TanStack Routerなど代替への移行議論」も見られるが、セキュリティパッチ適用が即時優先事項。

### Tailwind CSS + shadcn/ui
**Tailwind v4 + shadcn/ui**が2026年のSaaS/UIスタックのデファクトとして定着。Tailwind v4のRustエンジンによる高速ビルド（v3比2-10x）とCSS変数活用が好評。shadcn/uiは「所有するコンポーネント」としてカスタマイズしやすく、Radix UI + Tailwindベースでアクセシビリティ確保。

X投稿例：
- [@umeshchw](https://x.com/umeshchw/status/2055880178363228635): プロジェクト選択で「I will Go with ShadCN」と明言。
- [@Taniyatweets_](https://x.com/Taniyatweets_/status/2055866122394914844): 「Shadcn/UI made everyone feel like a designer overnight」とデザイナー体験の高さを指摘。
- [@soulblocks](https://x.com/soulblocks/status/2055873959795773872): shadcn関連パッケージを積極インストール。

カスタマイズTipsとして、wrapperパターンやCSS変数（--primaryなど）活用が推奨されている。

### React Aria Components
Adobeの**React Aria v1.17.0**（4月リリース）が最近の議論で言及。Expandable rows in Table、Virtualizerのwindow scrolling、horizontal virtualization追加。依存関係を大幅削減（90%減）し、tree shaking向上。アクセシビリティ重視の基盤としてTinybiguiなどライブラリで活用。

Xでは[@devongovett](https://x.com/devongovett/status/2055683397872500861)（React Aria開発者）が採用状況を共有。Base UI比較の声も。

### Prisma
Prisma 7.x（現在7.8.0）が安定運用中。**Prisma Next**（TypeScriptフルリライト）のロードマップ進行中：カスタムコレクションクエリ、streaming、SQL escape hatch、拡張システム（pgvector例）など。Prisma Postgres連携も強化。

Xではドキュメント古株問題の指摘あり（[@US_Khokhar](https://x.com/US_Khokhar/status/2055881367758709205)）。Drizzle/ZenStack比較議論も活発だが、Prismaのエコシステム優位性は継続。

### Biome
**Biome**（Rust製linter/formatter）がESLint + Prettier代替として勢力拡大。v2でtype-aware linting追加、Tailwind/CSSサポート強化。2026 RoadmapではHTML/Vue/Svelte/Astro対応、cross-language rulesを推進中。10-100x高速化が魅力。

X/ネットで「単一ツールで完結」「Prettier互換性97%」が高評価。実務移行例が増えている。
