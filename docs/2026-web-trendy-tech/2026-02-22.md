- [@ZamDeveloper](https://x.com/ZamDeveloper/status/2025137417888694600): 2025年初頭80%だったPrisma求人が2026年2月で55%に。Drizzle30%。
- [@DrizzleORM](https://x.com/DrizzleORM/status/2025268998426824876): Prisma + Drizzleのハイブリッド使用を「正しい方法」と肯定。
