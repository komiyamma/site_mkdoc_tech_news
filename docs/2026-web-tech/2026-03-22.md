**ここ1週間のWeb技術ホットトピックまとめ（2026年3月15〜22日頃）**

X（Twitter）の全ユーザー投稿とネットニュースから、指定トピックを中心に集めました。古い情報は除外し、最近のリリース・議論・トレンドに絞っています。AI統合やパフォーマンス向上、ベクトルDB活用が特に目立っています。

### React / Next.js
Next.js 16.2が3月18日にリリースされ、AIエージェント向け大幅強化が話題。`create-next-app`でデフォルトAGENTS.md生成（AIが公式ドキュメントを即参照可能）、ブラウザログをターミナル転送、Dev Serverロックファイル、実験的Agent DevTools（React DevToolsインスペクション・PPR解析・スクリーンショット取得）など。AI開発ワークフローが劇的に改善。

XではSvelteKitとの比較や実装例がちらほら（React単体は静かめ）。

### Vue / Svelte
- **Svelte**：3月アップデート（Svelte 5.50〜5.53 / SvelteKit 2.51〜2.53）。プログラム的コンポーネントで`createContext`使用可能、{@html}にTrustedHTML対応、HTMLタグ内コメント許可、サーバーサイドエラーバウンダリ追加。SvelteKitはナビゲーション時のscroll位置取得、Vite 8対応、Netlify Adapter更新。State of JS 2025でリアクティブフレームワーク1位継続。

X例：[@acoyfellow](https://x.com/acoyfellow/status/2035543139444773060)（SvelteKit + shadcn-svelteでクラウドシェル再構築、ファイルツリー・マルチタブ対応を報告）

- **Vue**：Vue.js Amsterdam 2026（3月開催）で大発表。Nuxt 4.4（Vue Router 5統合・DX/パフォーマンス向上）、Vue 3.6 Vapor Mode（仮想DOM除去で高速化、ベータ継続）、Vite 8（rolldown採用で爆速ビルド）。Vite Plus / Void Cloudも登場。
