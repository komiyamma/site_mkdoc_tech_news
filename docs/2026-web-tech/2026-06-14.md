### データベース / ベクトル検索
- **HelixDB**（Rust製graph-vector DB、object storage基盤）がHNで話題。グラフ+ベクトル+KVを1つで扱い、エージェントメモリ向けとして注目（NVIDIA/Vercelバック、YC）。スケーラビリティが高評価。
- **chDB**がHNSW vector_similarity index + MySQL/MongoDB table functions追加。埋め込みエンジンでベクトル検索+直接クエリ可能に。
- **Bun**の組み込みSQLite + vector検索がローカルで高速（50k docsでsub-20ms）と実践報告。
- **RisingWave**が「streaming-native vector retrieval」を提唱。従来ベクトルDBの鮮度問題（バッチ更新遅延）をストリーミングで解決する動き。
- Milvus/Qdrantなどの既存vector DBも継続的に言及され、AIアプリ基盤として安定人気。
