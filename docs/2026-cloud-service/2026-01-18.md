この一週間（2026年1月11日以降）の**Firebase**、**Vercel (v0を含む)**、**Cloudflare**、**Supabase**、**Azure**に関する新情報を、公式発表・ブログ・X投稿を中心にまとめました。古い情報は除外し、最新のものに絞っています。主なソースは公式リリースノート、ブログ、Xの開発者投稿です。

### Firebase
- **Firebase CLI v15** がリリース。Java 21+ がエミュレータ実行に必須となり、Cloud Functionsのタイムアウト検証が厳格化、Data ConnectがデフォルトでPostgres 17に。breaking changesを含むため、アップデート時は注意が必要。
- **Project IDX がFirebase Studioに統合**。既存ワークスペースはFirebase Studioで開けるようになり、フルスタックアプリ開発ツールとして強化。
- Geminiモデル一部の廃止予定（gemini-2.0-flash-001など、2026年2月〜3月）。Firebase AI Logic/Genkit使用者は新しいモデル（例: gemini-2.5-flash-lite）への移行を推奨。
- XではSupabase vs Firebaseの議論が活発だが、新機能関連の言及は少なく、主に比較投稿（例: [@vivoplt](https://x.com/vivoplt/status/2012491762023940503) のポール）。

### Vercel (v0を含む)
- **v0でAWSデータベース（Aurora PostgreSQL、Aurora DSQL、DynamoDB serverless）がネイティブ利用可能**。自然言語プロンプトでデータベースをプロビジョニング・接続でき、Vercel Marketplace経由で新アカウントに$100クレジット付与。フルスタックアプリを数分で構築可能に。
- **Vercel AI GatewayでClaude Code利用可能**。Vercel agentがコーディングガイドラインを自動検出・適用。
- **Secure computeがEnterpriseチーム向けself-serve**に。v0 × Sanity builder challenge開始（$3,000相当のv0クレジット懸賞）。
- Xでv0関連の投稿が多く、AWS統合が話題（例: [@v0](https://x.com/v0/status/2012211354795704538) の発表）。
- コミュニティニュース（1/2〜1/9週まとめ）でこれらのアップデートがハイライト。

### Cloudflare
- **SQLite-backed Durable Objectsのストレージ課金が2026年1月（最速1/7）から有効**。対象ユーザーは事前確認を。
- **ダッシュボードのリソース複製機能**追加（テーブルからDuplicate可能）。Terraformプロバイダv5移行ツールが3月に予定。
- **ネットワークメンテナンス**（IAD, YYZ, JAXなど複数データセンターで1月中旬に予定）。TurnstileのHTTP接続問題調査中。
- 過去の障害教訓からHealth Mediated Deployments (HMD) を2026年Q1までに全プロダクション適用予定（構成管理の安定化）。
- Xでは主にメンテナンスやセキュリティ関連の議論だが、新機能の目新しさは少ない。

### Supabase
- **Postgres 15の拡張機能サポートが終了予定**（Postgres 15のEOLが2026年5月頃）。既存プロジェクトは影響なしだが、新規/更新は避ける。
- **Developer Update - January 2026** で2026年のセキュリティ強化・ロードマップ公開。ダッシュボードのStorage UI刷新（Analytics/Vectorsバケット追加）。
- XでSupabase vs Firebaseの比較投稿が多く、Supabase推しが多い（例: [@vivoplt](https://x.com/vivoplt/status/2012491762023940503) のポール結果傾向）。
- オープンソースアップデート（過去1ヶ月まとめ）でMCP統合、Stripe自動同期、クエリExplainタブ、APIトラフィックマップなど多数追加。

### Azure
- **ExpressRoute 400G対応**が2026年から一部ロケーションで開始（マルチテラビットスループット可能）。VPN Gatewayが高スループット版GA（単一TCPフロー5Gbps、合計20Gbps）。
- **Standard NAT Gateway V2** がパブリックプレビュー（ゾーン冗長で追加コストなし）。
- CES 2026でNVIDIA Rubin Platform対応発表。Ignite 2025のAI/エージェント関連ニュースのフォローアップ（Foundry拡張など）が継続中。
- Xでは新機能の言及少なく、ゲーム/一般投稿が主。

全体的にVercel/v0のAWS統合とFirebaseのCLI更新が目立つ一週間でした。SupabaseとFirebaseの比較議論がXで継続中です。詳細は各公式ブログやXアカウントを確認してください！
