この一週間（2026年2月8日〜15日頃）の主な新情報や話題を、Firebase、Vercel（v0含む）、Cloudflare、Supabase、Azureごとにまとめました。主に公式リリースノート、ブログ、財務発表、X上の開発者投稿から抽出。古い情報は除外し、最新のものに限定しています。

### Firebase
- **SDKリリースとFirebase AI関連の更新**（2月9〜12日頃）：  
  Flutter SDK v4.9.0リリース（Firebase AI Logicの変更、バグ修正）。Unity SDK v13.8.0リリース（Firebase AIの新機能）。Android BoMの更新も含む。  
  Firebaseの公式リリースノートで確認可能。

- **Cloud Storageの価格変更関連**：  
  2026年2月3日からデフォルトバケット（*.appspot.com）のアクセス維持にBlazeプラン（pay-as-you-go）必須。無料プランでは新規プロビジョニング不可。移行を促すアナウンスが継続中。

- X上の話題：主にFirebase vs Supabaseの比較議論が多く、認証やモバイル開発での選択でFirebaseを推す声（安定性、無料枠の使いやすさ）が見られたが、新機能の具体的な言及は少なめ。

### Vercel（v0含む）
- **v0のメジャーアップデート**（2月3〜10日頃）：  
  AIアプリビルダーv0が大幅刷新。Gitワークフロー、セキュリティ強化、実プロダクション統合、GitHubリポジトリインポート、環境変数自動引き込みなど。エンタープライズ向けに進化し、4百万ユーザー超の成功を基に「vibe coding」を本格化。  
  新v0紹介ブログで詳細公開。コミュニティで「Free0」キャンペーン（有料ユーザー向け$10クレジット/日、2月上旬限定）も話題に。

- **その他のVercel更新**：  
  ChangelogでBrowserbase（AIエージェント向けブラウザ自動化）がMarketplace追加、MiniMax M2.5モデル対応、stale-if-errorキャッシュサポート（2月13日）、脆弱なnext-mdx-remoteバージョンのデプロイブロックなど。

- X上の話題：新v0の興奮が続き、クレジットキャンペーンやStripeの事例（飛行機内でアプリ完成）が共有された。

### Cloudflare
- **財務発表と業績**（2月10〜13日頃）：  
  2025年Q4および通年決算発表。Q4売上$614.5M（前年比+34%）、通年$2.17B（+30%）。2026年売上見通し$2.79〜2.80B（アナリスト予想上回る）。AI需要がクラウドサービスを牽引。株価急騰。

- **Markdown for Agents**：  
  エージェント向けにHTMLをMarkdownへリアルタイム変換（Content Negotiation Headers使用）。トークン80%削減、AIエージェントの効率化。ブログで公開。

- X上の話題：Markdown for Agentsが開発者間で拡散。AIエージェントのコスト削減として注目。Moltworker/OpenClaw関連の議論も。

### Supabase
- **重大インシデント**（2月12日）：  
  us-east-2（Ohio）リージョンで大規模障害（21:12 UTC開始、3時間42分）。Postgres、Auth、Storage、Realtimeなど全サービス影響。4.92%の顧客に影響。詳細ポストモーテム公開、再発防止策を記載。

- **その他の更新**：  
  PrivateLink GA（AWS VPC Lattice経由のプライベート接続）。Claude公式コネクタ（自然言語でDB管理）。Postgres AIエージェント向けベストプラクティス公開。pg_graphqlのデフォルト無効化（新プロジェクト、既存ゼロ使用プロジェクト対象）。

- X上の話題：障害が大きく話題に。ダウンタイム中のサインアップ不可で代替（Firebaseなど）議論。Claudeコネクタの利便性を称賛する声も。

### Azure
- **Azure Update（2月6〜13日頃）**：  
  John Savillの週次動画で複数更新：App GW DRS 2.2（OWASPルールセット更新）、AMD v6 confidential VMの新リージョン追加、Fleet managerのnamespace scope placementなど。  
  Azure SQLのVisual Studio Code統合強化（SQL Projects Publish dialog）。Thailand southリージョン関連も。

- **その他の話題**：  
  West USリージョンで電力障害（2月7〜8日）、Windows Update影響。BizTalkからLogic Appsへの移行ツールなど。

- X上の話題：週次アップデート動画が共有され、開発者間で議論。

全体として、Cloudflareの財務好調とSupabaseの障害が目立つ週でした。Vercel v0の進化も開発者コミュニティで活発。必要に応じて公式ブログやstatusページを確認してください。
