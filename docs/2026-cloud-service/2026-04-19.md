**Firebaseの新情報（2026年4月12〜19日）**  
Firebase Admin Dart SDKが4月16日に実験的リリースされました。Authentication、App Check、Cloud Firestore、Cloud Functions、Cloud Messaging、Security Rulesをサポートし、Dart開発者がサーバーサイドでFirebaseをより簡単に扱えるようになりました。  
CLI v15.14.0（4月16日）もリリースされ、プロジェクトのプログラム管理が強化されています。  
Cloud FirestoreではコンソールにAI支援機能が追加され、Mutationsにadd/remove演算子が新しく利用可能になりました。また、Remote ConfigがWebアプリのサーバーサイドSDKに対応し、SSR時の初期ロード性能を最適化できます。  
Android開発者向けには、Firebase AI Logicのハイブリッド推論（オンデバイス＋クラウド）と新Geminiモデル（画像生成向けNano Bananaなど）が追加され、アプリ内AI機能がさらに進化しています。

**GCP（Google Cloud Platform）の新情報（2026年4月12〜19日）**  
4月17日のリリースノートを中心に各種バグ修正と機能改善が公開されました（VS Code関連のクォータ修正など）。  
4月16日には複数の企業連携が発表され、AvidとのAgentic AIを活用したメディア制作連携、AEON360とのAI駆動型エコシステム、Thoma Bravoとの企業ソフトウェアAI化、Technogymとの健康・ウェルネスAI化などが進んでいます。  
Google Cloud Next関連の技術セッション情報も共有され、Geminiの最新動向や開発者ワークフローのAgent orchestrationが注目されています。

**Vercel（v0含む）の新情報（2026年4月12〜19日）**  
4月17日にデプロイ保持ポリシーが更新され、アクティブなブランチデプロイメントをより安全に保持できるようになりました（短い保持期間でもプレビューが失われにくく）。  
v0については、@rauchg CEOが「デザインを自律化（DESIGN.md）し、v0 Platform API/Sandboxを活用した次世代Agentがブランドシステムを自動維持する時代」と指摘。FigmaやClaudeとの役割分担を超えた抽象化レイヤーとしての進化を強調しています。

**Cloudflareの新情報（2026年4月12〜19日）**  
「Agents Week 2026」（4月13日開始）でAgent Cloudが大幅拡張されました。  
主な新機能：  
- **Agent Readinessツール**（isitagentready.com）：サイトがAI Agentに最適化されているかをスコアリング（discoverability、Markdown negotiation、llms.txt、MCPサーバー対応など）。  
- Dynamic Workers（AI生成コードをセキュアに高速実行するIsolateランタイム）。  
- Artifacts（Agent時代向けGit互換ストレージ）。  
- Shared Dictionaries（圧縮効率向上）、AIトレーニング向けRedirects（canonical URLへの301自動転送）など。  
AI Agent対応が本格化する一大アップデートです。

**Azureの新情報（2026年4月12〜19日）**  
**Smart Tier**がGA（Generally Available）になりました。Azure Blob Storage / Data Lake Storage (ADLS) で自動階層化がほぼ全ゾーン公開クラウドリージョンで利用可能に（Israel Central、Qatar Central、UAE Northを除く）。  
その他4月17日頃：Azure Databricks上でAnthropic Claude Opus 4.7がGA、AKSバックアップを単一Azure CLIコマンドで設定可能、Azure NetApp Filesのユーザー/グループクォータレポート追加など、ストレージ・コンテナ・AI関連の安定化・利便性向上が目立ちます。
