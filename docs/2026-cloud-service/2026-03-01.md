この一週間（2026年2月22日〜3月1日頃）のFirebase、Vercel（v0を含む）、Cloudflare、Supabase、Azureに関する新情報を、X（Twitter）の投稿やウェブソースから集めてまとめました。古い情報は除外し、主に最新の発表、機能更新、問題、議論を中心にしています。

### Cloudflare
Cloudflareはこの週にいくつかの注目すべき更新と出来事がありました。

- **Markdown for Agents**機能のリリース：AIエージェントがウェブページをリクエストすると、自動でHTMLをMarkdownに変換して提供。トークン使用量を最大80%削減可能。Claude CodeやOpenCodeなどのツールが即座に活用可能。
- **グローバル障害発生**：2月20日頃にBYOIPサービスで6時間の大規模障害（内部設定更新によるバグ）。一部の顧客に影響したが、サイバー攻撃ではなく復旧済み。
- **Vinextプロジェクト**：Claude AIを使って1週間でNext.jsをViteベースに再実装した実験的プロジェクト。ビルド4.4倍高速、バンドル57%小型化。Next.jsのAPI表面94%カバーと主張されるが、Vercel側から「本当のテストカバレッジは低く、部分的なサポートに過ぎない」と批判あり。
- その他：Workersの新機能（no config deploy、sandbox snapshots、pywrangler Windows対応、リソース上限引き上げ、Agents SDK 0.6.0）など、頻繁に更新中。

Xでは、Vinextの議論が活発で、CloudflareのAI活用アプローチが話題に。

例: [@dok2001](https://x.com/dok2001/status/2027934625557815398)（Cloudflare CTO）がVinextを紹介。

### Vercel（v0を含む）
Vercelの更新は比較的静かですが、コミュニティ投稿でv0関連の言及あり。

- WebStreamsをAIで10倍高速化、再構築。
- AI Gatewayが複数ビデオ生成モデル対応。
- Vercel Blobで機密ファイルのプライベートストレージ追加。
- Teams向けに使用量・コストデータのプログラムアクセス可能（FinOps統合）。

v0は引き続き生成AIでReactコンポーネント作成ツールとして人気。Xでは「v0でmeme coinサイト作成」など実例投稿あり。

VercelはCloudflareのVinextに対して移行ガイドを公開したが、メリット強調は控えめで「ロックイン」批判も。

### Supabase
最大のトピックは**インドでのアクセスブロック**問題。

- インド政府によりSupabaseがブロックされ、多くの開発者・スタートアップに影響。Supabaseチームは当局と交渉中、DNS変更やVPNを一時回避策として推奨。継続的に更新を投稿中。
- Xでは「Supabase vs Firebase/Postgres/MongoDB」のアンケートや議論が多く、Supabase推しが多いが、ブロック問題で懸念の声。
- その他：LovableなどのツールでSupabase統合のレビューあり。

例: Supabase公式アカウントが複数回更新を投稿。

### Firebase
Firebaseの目立った新発表は少ないが、関連議論あり。

- Xで「FirebaseよりPostgreSQL/Supabase推奨」の声多数。リアルタイム機能のデバッグが面倒という意見。
- Google IO関連の過去投稿でFirebase Studio（プロンプトでサイト構築）が言及されるが、この週の新情報は限定的。

全体的に「トレンドだからFirebase」ではなく、Postgresベースを勧める投稿が目立つ。

### Azure
Azureの更新は主に公式ブログやQ&Aから。

- **Team Calendar extension**のビジュアル更新（モダンデザイン、読みやすさ向上）。
- AKS Windows Server Annual Channelの廃止予定。
- App GW WAF Insights、Prem SSDv2の新可用性ゾーン、BlobのEntra ID対応SASなど。
- Azure AI Searchの感度ラベルサポート、PostgreSQLのgeoバックアップ強化。
- 新OpenAIモデル追加、Azure Monitorパイプライン更新など。

XではAzure関連の投稿は少なく、障害やサインインエラー（interaction_required）のQ&Aが散見。

全体として、この週はCloudflareの積極的なAI/Workers更新とSupabaseのインドブロック問題が最もホット。Vercel/Cloudflareの競争も注目点です。
