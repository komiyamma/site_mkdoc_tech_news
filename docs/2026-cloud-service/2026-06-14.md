**Firebase / GCP の主な新情報（過去1週間）**

- Firebase Admin Node.js SDK v14.0.0 がリリースされ、Node.js 22以上必須に。Cloud Messaging、Authentication、Cloud Functions、Firestore などの改善とエラーハンドリング強化。
- Agent Skills に Crashlytics と Remote Config が追加され、IDE内デバッグ支援が強化。Firebase チームの公式投稿より。[@Firebase](https://x.com/Firebase/status/2065494556368392241)

**Supabase の主な新情報**

- **Developer Update - June 2026** 公開（6月6日頃）：Series F で $500M 調達（$10B valuation）、Multigres 0.1 alpha リリース（Postgres のスケーラブル OS）、Passkey Sign-in、ChatGPT アプリ統合、AI Coding Agents プラグインなど。
- iceberg-js v1.0.0 リリース。[@supabase](https://x.com/supabase/status/2065442757460500813)

**Vercel（v0含む）の主な新情報**

- Blob store limit が Hobby ユーザーで増加（5→100）。Workflow SDK が Nitro v3 でネイティブ対応など最近の changelog 更新。
- Drop（drop.new）リリース：ブラウザでファイルをドラッグするだけで即時デプロイ・プロジェクト作成。AI時代の手軽なプロトタイピングに好評。

**Cloudflare の主な新情報**

- Security Insights が 120 scans/sec 以上処理可能に（10x スケール）。Kafka/Postgres 最適化でハード追加なし。[@Cloudflare](https://x.com/Cloudflare/status/2065420105727520949)
- Mastercard との連携強化（Agent Pay 関連）。

**Azure の主な新情報**

- 週次更新で Azure Boost Guest RDMA（Preview）、Premium SSD v2 非ゾーン対応（GA）、NCv6（GA）、PostgreSQL 関連改善、Managed Redis Entra RBAC など複数機能。[@NTFAQGuy](https://x.com/NTFAQGuy/status/2065456829979929947) などの議論活発。

**全体の傾向（比較・エコシステム）**  
Supabase vs Firebase の議論が継続（SQL vs NoSQL、pricing、AI統合）。Vercel/Supabase/Cloudflare の無料ティア組み合わせが「$0インフラ」として注目されるケースも。GCP/Firebase は AI Logic や Agent Skills でモバイル・フルスタック強化を進めている。
